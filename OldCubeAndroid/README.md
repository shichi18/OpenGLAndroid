# OldCubeAndroid
This is OldCubeAndroid

作成日：2018/4/21  
更新日：2018/4/22  

使用言語：Java

AndroidでOpenglを使った練習テストとして作成。  
OpenGLESのバージョンを下げて練習。
立方体をAndoroidに表示すること目標。 
光源設定、カメラ設定の確認


## 開発環境/実行環境
AndroidStudio 3.1  
OpenGL ES 1.0  
HUAWEI HDN-W09 (Android 7.0,API 24)
